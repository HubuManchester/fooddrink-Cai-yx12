# Random Recipe Recommender

A .NET MAUI cross-platform mobile app for discovering random recipes.

## Features
- Browse, search and filter recipes
- Add to favorites
- Shake phone for random recipe
- Text-to-speech for recipe steps
- Take photos of your cooked dishes
- Dark mode and large text mode

## Hardware Used
- Accelerometer (shake detection)
- Text-to-Speech
- Camera
- Location

## How to Run
1. Open in Visual Studio 2022
2. Select Android emulator
3. Press F5
